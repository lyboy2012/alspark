CocoaPods 是什么？

CocoaPods is a dependency manager for Swift and Objective-C Cocoa projects.

是个swift oc Cocoa 项目依赖管理(前端等价于 npm yarn 包管理工具)

Cocoa是苹果公司为Mac OS X所创建的原生面向对象的API，是Mac OS X上五大API之一（其它四个是Carbon、POSIX、X11和Java）
